    function

    La función toma tres parámetros:

    • $base: El valor base sobre el cual se va a aplicar el factor.
    • $factor: El factor multiplicador que ajustará la anchura base.
    • $unit: La unidad que se usará para la anchura calculada (por defecto, vw).

    La función retorna el valor calculado como una expresión que multiplica el valor base por el factor y luego le añade la unidad especificada.

    functions

    calcularAnchura

    @function calcularAnchura($base, $factor, $unit) { ... }

    Description

    Calcula la anchura basada en un valor base, un factor de multiplicación y una unidad.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $base

    El valor base de la anchura.

    		Number— none
    $factor

    El factor de multiplicación.

    		Number— none
    $unit

    (Opcional) La unidad que se usará para la anchura calculada. El valor predeterminado es vw.

    		String— none

    Returns

    String —

    La anchura calculada con la unidad especificada.

    rows

    $rows: #f11b0d,#ef0fe2,#11eedc,#e5ee11,#888377,#f0960e;

    Description

    Lista de colores utilizados para las filas de una tabla. Cada color en esta lista se utiliza para alternar el fondo de las filas en una tabla, proporcionando un diseño visualmente atractivo. Los colores son los siguientes:

    • Fila 1: #f11b0d (rojo)
    • Fila 2: #ef0fe2 (púrpura)
    • Fila 3: #11eedc (verde claro)
    • Fila 4: #e5ee11 (amarillo)
    • Fila 5: #888377 (gris)
    • Fila 6: #f0960e (naranja)

    Type

    Color

    variables

    width

    $width: 30vw;

    Description

    El ancho de los elementos, definido como un porcentaje del ancho de la ventana. Esta variable se utiliza para establecer el tamaño de imágenes y otros elementos que requieren un ancho dinámico. El valor se define en vw (viewport width), lo que significa que el ancho será un 30% del ancho de la ventana.

    Type

    Width
